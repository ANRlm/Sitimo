# MathLib 功能增强需求文档

**创建时间**: 2026-04-22  
**项目**: MathLib 数学题库管理系统  
**版本**: 1.0

## 需求概述

基于用户反馈，对 MathLib 系统进行以下功能增强：

### 1. 批量删除功能
**问题**: 图片和题目无法批量删除，操作效率低  
**需求**: 在图库和题库页面增加批量选择和删除功能

### 2. 示例数据系统
**问题**: 新用户无法快速体验系统功能  
**需求**: 
- 预置包含各功能的示例数据（题目、图片、试卷、标签）
- 提供一键删除所有示例数据的功能
- 示例数据需明确标识，避免与用户数据混淆

### 3. 学科字段简化
**问题**: 系统仅用于数学题库，学科选择冗余  
**需求**: 移除所有与学科选择相关的代码和 UI 组件

### 4. 年级范围扩展
**问题**: 当前年级选项不够标准化  
**需求**: 
- 标准化年级选项：初一、初二、初三、高一、高二、高三
- 历史数据保留但标记为"自定义"

### 5. 试卷留空行功能
**问题**: 试卷编辑时无法插入留空行供学生作答  
**需求**: 
- 在试卷编辑器中增加留空行插入功能
- 留空行高度可自由调整
- LaTeX 导出时正确渲染留空行

## 关键决策

### 示例数据标识策略
**方案**: ID 前缀 `demo-`  
**理由**: 
- 无需数据库 schema 变更
- 实现简单，查询高效
- 易于识别和过滤

### 批量删除被引用图片
**方案**: 允许删除，题目中显示占位符  
**理由**: 
- 用户拥有完全控制权
- 避免复杂的引用检查逻辑
- 占位符提供视觉反馈

### 年级值标准化
**方案**: 提供预定义列表，历史数据保留但标记为"自定义"  
**理由**: 
- 保持数据完整性
- 向后兼容
- 引导用户使用标准值

### 留空行单位
**方案**: 使用固定高度（1em = 当前字号），LaTeX 使用 `\vspace{Xem}`  
**理由**: 
- 与字体大小成比例
- LaTeX 标准语法
- 易于理解和调整

### 功能验收
- [ ] 图库和题库支持批量选择和删除
- [ ] 系统包含完整的示例数据
- [ ] 可一键删除所有示例数据
- [ ] 所有学科选择 UI 已移除
- [ ] 年级选项标准化为 6 个固定值
- [ ] 试卷编辑器可插入和调整留空行
- [ ] 留空行在 PDF 导出中正确显示

### 技术验收
- [ ] 数据库迁移脚本无错误
- [ ] API 端点测试通过
- [ ] 前端组件无 TypeScript 错误
- [ ] 批量操作性能可接受（< 2s for 100 items）
- [ ] 示例数据删除不影响用户数据

## 优先级

**P0 (必须)**:
- 批量删除功能
- 示例数据系统
- 留空行功能

**P1 (重要)**:
- 学科字段移除
- 年级标准化

## 风险评估

**低风险**:
- 示例数据系统（独立功能）
- 年级标准化（向后兼容）

**中风险**:
- 批量删除（需考虑性能和事务）
- 学科字段移除（影响多个模块）
- 留空行功能（需修改导出逻辑）
